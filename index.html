Swift Accelerator • 2025

Scripties

App Store
Visit Scripties ↗

The Problem

Many students, presenters, and professionals have a hard time presenting speeches confidently. They face stress due to script writing, last-minute rehearsals, and uncertainty about pacing and clarity.

The Solution

Scripties helps users write, rehearse, and improve their scripts efficiently. The app consolidates script management, teleprompter-style practice, recording, playback, and automated feedback on key delivery metrics.

Developed as part of the 2025 Swift Accelerator Program, Scripties is a mobile application helping students master public speaking through real-time feedback and intelligent practice tools. We overcame repeated design rejections and a grueling two-week coding sprint, navigating SwiftUI errors to launch on the App Store. The app tracks speech delivery using Words Per Minute (WPM) metrics, measures pacing consistency, and provides actionable feedback to help users improve their presentation skills.

BuildingBlocs • 2025

Sign Language AI

The Challenge

Communication barriers exist for the deaf and hard-of-hearing community. Traditional tools for sign language translation are often complex, expensive, or require specialized hardware.

The Innovation

An AI-powered system that detects sign language gestures and translates them into text in real-time using computer vision. Built to break down communication barriers with accessible technology.

Following an intensive machine learning course, we built an AI that fine-tuned an untrained model and integrated it into a responsive web interface using computer vision techniques. The system achieved high accuracy in gesture recognition across diverse lighting conditions.
